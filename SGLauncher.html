SGLauncher


Launch Apps, execute commands (in terminal and in background), fast web search and basic calculator functionality



Features:

  • Show system menu apps
  • Search On Web buttone
  • Run In Terminal button
  • Button to change Rotation of display
  • Show calc answers
  • Have quick access buttons
  • Have a simple configuration tool
  • Show Actions


Download:

Clone the repo and build with make



git clone --depth 1 https://github.com/SGDE/SGLauncher

cd SGLauncher
    
make
